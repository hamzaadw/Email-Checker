# 📧 Email Checker

A simple Python project that checks if an email address is valid using regular expressions. This can be useful for quick validation or integration into larger systems like sign-up forms or user authentication flows.

---

## 🚀 Features

- ✅ Validates email address format
- 🧠 Uses Python's built-in `re` module (regex)
- 🖥️ Command-line friendly

---

## 🛠️ Requirements

- Python 3.x
